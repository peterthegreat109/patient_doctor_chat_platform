# patient_doctor_chat_platform
该小程序为本科在校毕业设计，功能稍微单一，后续会进行功能拓展。
开发平台为VSCODE,微信开发者工具
后端为NODE.JS，采用websocket进行定向广播，实现一对一交流。
数据库为Mysql。
